Integrasi Google Spreadsheet

Aktifkan agar setiap tambah, update, atau hapus pengeluaran otomatis terkirim ke Google Spreadsheet.

Cara Pakai Integrasi

File ZIP ini sudah dilengkapi kode Google Apps Script. Copy file google-apps-script.gs ke Apps Script yang terhubung dengan Spreadsheet kamu, deploy sebagai Web App, lalu tempel URL Web App di form ini.

1. Buat Google Spreadsheet Buat spreadsheet kosong, lalu buka Extensions → Apps Script.
2. Pasang Apps Script Copy kode dari file google-apps-script.gs dan samakan Secret Key.
3. Deploy Web App Pilih Deploy → New deployment → Web app → Anyone.
4. Dashboard Otomatis Buka URL /exec untuk melihat dashboard modern dari Apps Script.
5. Tempel URL Masukkan URL /exec ke form integrasi, lalu aktifkan sinkronisasi.

Backup Data

Karena GitHub Pages tidak menggunakan database, data tersimpan di browser. Gunakan backup agar data bisa dipindahkan ke perangkat lain.
